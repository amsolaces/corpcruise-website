About Access Codes

What are Access Codes?
Unique codes that allow corporate clients to verify their identity before booking rides. This prevents fake bookings from retail customers.

How it works:

  1. 1. User clicks "Book Ride"
  2. 2. First step asks for access code
  3. 3. If code is valid → Booking form unlocks
  4. 4. If code is invalid → Error message

Code Format:
CORP-[COMPANY]-[YEAR]-[NUMBER]
Example: CORP-ABC-2024-001

Expiry & Usage:
Set expiry dates and booking limits per code for better control.
